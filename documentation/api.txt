# endpointove za user related operacii v /user

login/
post request s username i pass, loginva user

logout/
get request s pk na usera, logoutva user

create/
post request pravi user 

get/
get request s pk vzema public infoto na user razlichen ot lognatiq

me/
get request s pk vzema vsichkoto info za lognatiq user

update/
put request s pk updateva infoto na user

follow/
get request (mai?) s pkove na usera koito followva i biva follownat

delete/
delete request s pk deleteva user
